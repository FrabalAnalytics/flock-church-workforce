# Flock commercial analysis

Generated on 22 July 2026 from current official supplier pricing and explicit planning assumptions.

- `Flock_Commercial_Analysis_Without_WhatsApp.docx` covers the core Flock platform with manual care follow-up.
- `Flock_Commercial_Analysis_With_WhatsApp.docx` covers the platform with metered Twilio/Meta WhatsApp delivery.

The figures are planning estimates rather than supplier quotations or financial advice. Recheck USD/NGN, supplier pricing, tax treatment, payment-processing fees, staffing, and actual message category before signing customers.

Regenerate both documents from the repository root:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/generate-commercial-documents.ps1
```
